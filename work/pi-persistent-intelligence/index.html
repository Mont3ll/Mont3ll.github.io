PI Persistent Intelligence

Governed memory and session search for AI coding agents.

Role

Live site →Source code →

A TypeScript package that gives PI coding agents durable, searchable memory across sessions. I designed the canonical JSONL record format, daily and long-term memory layers, inbox promotion flow, contested-memory warnings, session search, recall diagnostics, and dual-registry package publishing. The project is relevant to agentic product work because it treats AI memory as governed infrastructure rather than loose notes.
